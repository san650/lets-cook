# Lets-cook

Let's cook is your chef companion. Plan your weekly meals and know what to buy and how to cook the meals you want.

## Development

### Installation

```sh
$ git clone https://github.com/san650/lets-cook.git
$ cd lets-cook
$ npm install
$ bower install
$ ember server
```

Then visit http://localhost:4200

### Tests

```sh
$ ember test --server
```
